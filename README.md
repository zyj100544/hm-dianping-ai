# Heima Dianping

## Startup

### Backend (Spring Boot)

```bash
cd hm-dianping
mvn spring-boot:run
```

### Frontend (Vue 3 + Vite)

```bash
cd hm-dianping-web
npm install
npm run dev
```

## Prerequisites

- Java 8
- Maven
- Node.js and npm



$env:JAVA_HOME="D:\develop\jdk\jdk_1.8"; mvn spring-boot:run
